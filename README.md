# QuantiCore 📊

### A Smart System for Quantitative Data Analysis

QuantiCore is an advanced **Streamlit-based web application** designed to simplify quantitative data analysis through an interactive and user-friendly interface. It enables users to upload CSV datasets and perform preprocessing, visualization, statistical analysis, machine learning regression, and hypothesis testing — all in one platform.

---

## 🚀 Features

### 📂 Data Upload & Dashboard

* Upload CSV datasets easily
* Display:

  * Number of rows
  * Number of columns
  * Missing values
* Interactive preview of dataset

### 🛠️ Data Preprocessing

* Handle missing values automatically
* Encode categorical variables
* Normalize numerical features
* Session state data persistence

### 📈 Data Visualization

* Histogram with KDE
* Boxplot
* Scatter plot with regression trendline

### 🎯 Sampling Methods

* Random Sampling
* Stratified Sampling

### 🔗 Correlation & Regression

* Simple Linear Regression (SLR)
* Correlation coefficient calculation
* Heatmap generation
* Partial Correlation
* Multiple Correlation

### 🤖 Machine Learning

* Multiple Linear Regression (MLR)
* Actual vs Predicted visualization
* OLS statistical model

### 📊 Maximum Likelihood Estimation (MLE)

Supports:

* Normal Distribution
* Poisson Distribution
* Exponential Distribution

### 🧪 Hypothesis Testing

* Z-Test
* T-Test
* Statistical significance decision support
